# ElectroStore Admin Dashboard

React + Vite admin panel for ElectroStore e-commerce platform.

## Setup

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and set `VITE_API_URL`
3. Run dev server: `npm run dev`

## Features

- Authentication (login, protected routes)
- Dashboard with stats
- Product management (CRUD + images)
- Promo management
- Order management with status updates
- Transaction viewing
- Customer list
- Staff user management
- System settings
- Notifications
- Reports (sales, top products)
- Print invoices and reports

## Build

`npm run build` – outputs to `dist` folder.
